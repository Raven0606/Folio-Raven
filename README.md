# 🎵 [SPOTMUSIC]

![License](https://img.shields.io/badge/license-MIT-blue.svg)

## 📖 Overview
[SPOTMUSIC] is a simple Music Player Website built for the purpose of student project, it allows you to easily manage, play, and enjoy their favorite local audio tracks.

## ✨ Features
* **Core Playback:** Play, pause, skip, and rewind functionalities.
* **Playlist Management:** Create, edit, and organize your favorite tracks.
* **Dynamic UI:** A beautiful, responsive interface that adapts to your screen size.
* **Seek Controls:** Intuitive sliders for volume adjustment and track navigation.
